# CookBook
## Приложение находится в стадии разработки

## Краткое описание приложения
### Приложение позволяет хранить кулинарные рецепты и выводить список продуктов и необходимое их количество

## Более полное описание приложения
### Допустим Вы бедный и вам нужно купить продуктов на неделю, не потратив лишних денег. Или Вы нормальный человек и Вам нужно закупить много продуктов для торжества (тщеславия...). В любом случае Вам нужно определиться с перечнем блюд, выбрать список продуктов с количеством для каждого блюда, посчитать количество в совпадающих продуктах и оформить в понятный список. Даже при небольшом количестве блюд работа слегка утомительна.....С помощью приложения Вы выбираете "стол", нажимаете кнопку и получаете готовый список.

## Когда приложение будет готово пользователь сможет:  
### Cоздавать и редактировать рецепты блюд.
### Cоздавать и редактировать из блюд "столы"
### Выводить на экран список продуктов и посчитанное количество продуктов.

## При разработке используются
### PHP 8.2 
### Node.js v22.19.0
### Laravel 11.3.1

## Как загрузить приложение:
### Скачать архив из репозитория, извлечь архив в выбранную папку
### В терминале переходим в папку с проектом: cd your-project
### Устанавливаем PHP-зависимости (Laravel установится локально): composer install
### Устанавливаем JavaScript-зависимости: npm install
### Копируем файл окружения: cp .env.example .env 
### Выполняем команду генерации ключа: php artisan key:generate
### Запускаем миграции: php artisan migrate
### Собираем фронтенд: npm run build
### Запускаем сервер: php artisan serve
